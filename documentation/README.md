# Documentation

This folder contains **only the final academic deliverables**:

| File | Description |
|------|-------------|
| `Memoire_Final.tex` | Mémoire PFE aligné Guide IARO (Introduction → État de l'art → Méthodologie → Implémentation/expérimentation → Analyse → Conclusion) |
| `Memoire_Final_pre_IARO_backup.tex` | Sauvegarde avant restructuration IARO |
| `figures/` | Captures d'écran intégrées au mémoire (dossier `screen/` à la racine) |
| `Memoire_references.bib` | Companion bibliography extract |
| `Scientific_Article_Final.tex` | Scientific article (IEEE Access class) |
| `Scientific_Article_references.bib` | Article bibliography (copy) |
| `references.bib` | Bibliography used by `\bibliography{references}` |
| `IEEEtran.bst` | IEEE bibliography style |

Drafts, Overleaf ZIP exports, Jira/Kanban annexes, and intermediate TeX revisions were removed on purpose.
